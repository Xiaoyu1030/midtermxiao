Welcome to Ukulele Chords!

This page brings you the most-commonly-used ukulele chords. Default tuning is GCEA(standard or C tuning) but you can simply select one of the quick tunings，ADF#B (soprano or D tuning) and DGBE (baritone tuning) to change it immediately.

After you have selected your tuning you need to first select a root note (e.g. if you are looking for the Am chord, the root note is 'A'). The final step is to then select one of the many chord types available, all of the popular ones are included as well as some more exotic ones. Let's have a try and see how this ukulele is going to change.
